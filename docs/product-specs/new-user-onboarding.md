# New User Onboarding (Draft)

## Objective

Get a new team from zero to first usable Seer insight quickly.

## MVP Onboarding Flow

1. Start platform locally.
2. Ingest Prophet local ontology Turtle file.
3. Validate and upsert ontology.
4. Ingest sample events.
5. Explore ontology in read-only UI.
6. Ask ontology questions via copilot.
7. Open process explorer and run first object-centric process view.
8. Run first root-cause analysis with user-defined outcome.

## Success Signals

1. User can complete the flow without manual database intervention.
2. User can interpret at least one AI-assisted insight with supporting evidence.
